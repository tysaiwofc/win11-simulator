Sobre o Projeto

O Windows 11 Simulator é um aplicativo desktop open source construído com Electron que recria fielmente a experiência do Windows 11. Diferente de uma versão web, esta implementação oferece:

  • Performance nativa como aplicativo desktop
  • Acesso a APIs do sistema quando necessário
  • Instalação como um aplicativo convencional
  • Funcionalidades mais profundas de sistema

Recursos Técnicos

⚡

Arquitetura Electron

Combina Chromium para renderização e Node.js para acesso ao sistema, proporcionando uma experiência híbrida web-desktop.

🔧

APIs Nativas

Acesse recursos do sistema como sistema de arquivos, notificações nativas e mais através das APIs do Electron.

📦

Empacotamento Multiplataforma

Disponível para Windows, macOS e Linux com auto-update integrado.

🖥️

Apps Simulados Funcionais

Calculadora, Editor de Texto e outros apps com funcionalidades reais implementadas.

🧩

Sistema de Plugins

Arquitetura modular que permite adicionar novos recursos através de plugins.

🔌

Integração com Sistema

Atalhos de teclado, menu de contexto e outras integrações com o SO hospedeiro.

Requisitos do Sistema

  • Windows: Windows 10/11 64-bit, 2GB RAM
  • macOS: macOS 10.10 (Yosemite) ou superior
  • Linux: Distribuições modernas com GLIBC 2.24+
  • Processador: Dual-core 1.5GHz+
  • Armazenamento: 200MB de espaço livre
  • GPU: Aceleração de hardware recomendada

Desenvolvimento e Contribuição

Como projeto open source, todas as contribuições são bem-vindas! Siga estes passos para começar:

Configurar Ambiente

git clone https://github.com/tysaiwofc/win11-simulator.git
cd windows11-simulator
npm install

Executar em Dev Mode

npm run dev

Usa Electron Forge com hot-reload

Empacotar para Produção

npm run build

Gera executáveis para todas as plataformas

Encontre issues marcados como "good first issue" para começar a contribuir ou sugira novos recursos!
