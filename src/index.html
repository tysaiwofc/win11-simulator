    Sobre

    Windows 11 Simulator é um projeto de código aberto criado com a intenção de replicar a experiência de desktop do Windows 11 , utilizando tecnologias padrão como Electron, CSS e JavaScript.

    Este projeto não tem qualquer afiliação com a Microsoft e não deve ser confundido com o Sistema Operacional ou Produtos da Microsoft.

    Este também não é Windows 365 Cloud PC.

    Microsoft, Windows e outros produtos demonstrados neste projeto são marcas registradas do grupo Microsoft.
